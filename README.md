# ShopVerse - Online E-Commerce Website

## Project Links

- Repository: https://github.com/dharmapatel-jsn/shopverse
- GitHub Pages: https://dharmapatel-jsn.github.io/shopverse/
- Deployed Client App: https://shopverse-five-psi.vercel.app/
- Deployed Admin App: https://shopverse-five-psi.vercel.app/admin

## Frontend Features

- Browse products by category like clothing, electronics, bags, and footwear.
- View product cards with image, name, and price.
- Add products to cart and update cart items.
- Use login and signup pages with form validation.
- Use demo login credentials when backend is not available.
- Place orders through a simple checkout flow.
- Access admin panel to manage products and view dashboard data.
- Use responsive pages that work on desktop and mobile screens.

## Backend Features

- Built with Node.js and Express for API routes.
- Uses MongoDB with Mongoose models for users, products, cart, and orders.
- Supports authentication with JWT-based login.
- Hashes passwords with bcrypt before storing them in the database.
- Uses middleware for auth checks, role checks, and error handling.
- Includes separate routes for admin, products, cart, orders, and customer actions.
- Provides dashboard and inventory endpoints for admin-side data.
